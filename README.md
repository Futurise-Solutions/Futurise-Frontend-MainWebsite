## Animation Principles

Avoid:

* Excessive motion
* Laggy effects
* Over-engineered interactions
* Animations that hurt performance

Focus on:

* Scroll-triggered reveals
* Subtle parallax
* Hover interactions
* Content transitions
* Micro-interactions
* Section entrances
* Premium motion design

Everything should remain smooth on desktop and mobile.
